Selbständiger Software Entwickler

Mein Name ist Jens Krause und ich bin selbständiger Software Entwickler mit Sitz in Hamburg und Scharbeutz.

Bereits seit 1999 habe ich beruflich mit unterschiedlichen Webtechnologien zu tun. Dabei spielten sogenannte 'cutting-edge' Technologien immer eine sehr wichtige Rolle. Heute sind es React und React Native sowie die zukünftigen Versionen von Angular 2 und Ionic 2.

Neben meiner Arbeit geniesse ich das Leben am Ostseestrand der Lübecker Bucht. Entweder mit dem Fahrrad, beim Joggen oder einfach nur auf den Händen...
